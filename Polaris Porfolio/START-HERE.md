## â“ Having Issues?

### "Node.js didn't install properly"
- Make sure you restarted your computer
- Try downloading again from nodejs.org
- Choose the **LTS version** (left button)

### "npm install is taking forever"
- This is normal! It downloads ~150MB of files
- Wait 3-5 minutes
- Don't close the window

### "The site won't open"
- Make sure you ran `npm run dev`
- Check if you see "Local: http://localhost:3000"
- Manually go to http://localhost:3000 in your browser

### "I see errors in the terminal"
- Copy the error message
- Check TROUBLESHOOTING.md
- Most common: forgot to restart after Node.js install
