## ðŸ”¥ **Want Firebase? Here's What to Do:**

### First, Install Required Tools:

**1. Install Node.js:**
   - Go to: https://nodejs.org/
   - Download the LTS version (left button)
   - Run the installer
   - Click Next, Next, Next, Install
   - **RESTART** your terminal/VS Code after installation

**2. Install Firebase CLI:**
   Open a NEW terminal and run:
   ```bash
   npm install -g firebase-tools
   ```

**3. Login to Firebase:**
   ```bash
   firebase login
   ```
   (This opens browser - sign in with Google)

**4. Initialize Firebase in your folder:**
   ```bash
   cd "c:\Users\Bandla_951807\Polaris Porfolio"
   firebase init hosting
   ```
   
   **Answer these:**
   - Use existing project OR create new one
   - Public directory? Type: `.` (just a dot)
   - Single-page app? Type: `N`
   - Overwrite portfolio.html? Type: `N`

**5. Deploy:**
   ```bash
   firebase deploy
   ```
